# Féléves Feladat Koncepció
Egy 3D-s Játék, amelyben a játékos az Alföldön kezdi a játékot. Egy földúton kezdődik a játék.
A játék során a földút mentén át kell sétálnia a helyi kisboltba, ahol a falu bölcs öregei
tanácskoznak, ugyanis elromlott Pista bácsi régi UAZ 452-es gépjárműve, és segítséget igényel.
Az UAZ kuplungja elfüstölt(itt lesz egy cutscene ahol az UAZ modell alól füst jön ki).
A falu bölcsei a játékos ségítségét kérik, hogy kijelölt pontokra sétálva gyűjtsön össze alkatrészeket.
Az alkatrészek kis téglalap alakú dobozokban lesznek. A séta során a játékos ki lesz téve az időjárás 
viszontagságainak(pl:eső), és több méter hosszúságú víztócsákon is át kell jutnia. 
Az alkatrészek összegyűjtése után a játékos visszatér a helyi kisboltba, ahol a bölcs öregek már várják.
Az alkatrészeket a játékos átadja a bölcs öregeknek. Ezután a játék vége egy cutscene, ahol az UAZ-452-es
gépjármű annyira boldog lesz az új alkatrészektől hogy elkezd le fel rúgózni.
